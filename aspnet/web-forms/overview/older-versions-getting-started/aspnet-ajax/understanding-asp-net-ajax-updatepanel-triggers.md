## <a name="triggers"></a><span data-ttu-id="128b6-118">*Tetikleyiciler*</span><span class="sxs-lookup"><span data-stu-id="128b6-118">*Triggers*</span></span>

<span data-ttu-id="128b6-128">Visual Studio 'da biçimlendirme düzenleyicisinde çalışırken, bir `UpdatePanel` denetiminin iki alt öğesi olduğunu fark edebilirsiniz (IntelliSense 'den).</span><span class="sxs-lookup"><span data-stu-id="128b6-128">When working in the markup editor in Visual Studio, you may notice (from IntelliSense) that there are two child elements of an `UpdatePanel` control.</span></span> <span data-ttu-id="128b6-129">En sık görülen öğe, genellikle güncelleştirme paneli tarafından tutulacak içeriği (kısmi işleme etkinleştiriyoruz içerik) kapsülleyen `<ContentTemplate>` öğesidir.</span><span class="sxs-lookup"><span data-stu-id="128b6-129">The most-frequently seen element is the `<ContentTemplate>` element, which essentially encapsulates the content that will be held by the update panel (the content for which we are enabling partial rendering).</span></span> <span data-ttu-id="128b6-130">Diğer öğesi `<Triggers>` öğesidir ve bu, &lt;tetikleme&gt; öğenin bulunduğu UpdatePanel denetiminin kısmi işlemesini tetikleyebilecek, sayfadaki denetimleri (veya kullanıyorsanız Kullanıcı denetimi) belirtir.</span><span class="sxs-lookup"><span data-stu-id="128b6-130">The other element is the `<Triggers>` element, which specifies the controls on the page (or the user control, if you are using one) that will trigger a partial render of the UpdatePanel control in which the &lt;Triggers&gt; element resides.</span></span>

<span data-ttu-id="128b6-131">`<Triggers>` öğesi iki alt düğümün her birini içerebilir: `<asp:AsyncPostBackTrigger>` ve `<asp:PostBackTrigger>`.</span><span class="sxs-lookup"><span data-stu-id="128b6-131">The `<Triggers>` element can contain any number each of two child nodes: `<asp:AsyncPostBackTrigger>` and `<asp:PostBackTrigger>`.</span></span> <span data-ttu-id="128b6-132">Her ikisi de iki özniteliği kabul eder, `ControlID` ve `EventName`ve geçerli kapsülleme birimi içinde herhangi bir denetimi belirtebilir (örneğin, UpdatePanel denetiminiz bir Web Kullanıcı denetimi içinde bulunuyorsa, Kullanıcı denetiminin bulunacağı sayfada bir denetime başvurulması gerekmez).</span><span class="sxs-lookup"><span data-stu-id="128b6-132">They both accept two attributes, `ControlID` and `EventName`, and can specify any Control within the current unit of encapsulation (for instance, if your UpdatePanel control resides within a Web User Control, you should not attempt to reference a Control on the Page on which the User Control will reside).</span></span>

<span data-ttu-id="128b6-133">`<asp:AsyncPostBackTrigger>` öğesi özellikle, yalnızca bu tetikleyicinin alt öğesi olan UpdatePanel 'ın değil, kapsülleme birimindeki *herhangi* bir UpdatePanel denetiminin alt öğesi olarak var olan bir denetimden herhangi bir olay hedefleyebileceği için yararlıdır.</span><span class="sxs-lookup"><span data-stu-id="128b6-133">The `<asp:AsyncPostBackTrigger>` element is particularly useful in that it can target any event from a Control that exists as a child of *any* UpdatePanel control in the unit of encapsulation, not just the UpdatePanel under which this trigger is a child.</span></span> <span data-ttu-id="128b6-134">Bu nedenle, kısmi sayfa güncelleştirmesini tetiklemek için herhangi bir denetim yapılabilir.</span><span class="sxs-lookup"><span data-stu-id="128b6-134">Thus, any control can be made to trigger a partial page update.</span></span>

<span data-ttu-id="128b6-135">Benzer şekilde, `<asp:PostBackTrigger>` öğesi kısmi sayfa işlemeyi tetiklemek için kullanılabilir, ancak sunucu için tam gidiş dönüş gerektiren bir tane.</span><span class="sxs-lookup"><span data-stu-id="128b6-135">Similarly, the `<asp:PostBackTrigger>` element can be used to trigger a partial page render, but one that requires a full round-trip to the server.</span></span> <span data-ttu-id="128b6-136">Bu tetikleyici öğesi, bir denetim normalde kısmi sayfa işlemeyi tetikleyeceğinden (örneğin, bir UpdatePanel denetiminin `<ContentTemplate>` öğesinde bir `Button` denetimi olduğunda) tam sayfa işlemesini zorlamak için de kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="128b6-136">This trigger element can also be used to force a full page render when a control would otherwise normally trigger a partial page render (for instance, when a `Button` control exists in the `<ContentTemplate>` element of an UpdatePanel control).</span></span> <span data-ttu-id="128b6-137">Yine, PostBackTrigger öğesi, geçerli kapsülleme birimindeki herhangi bir UpdatePanel denetiminin alt öğesi olan herhangi bir denetimi belirtebilir.</span><span class="sxs-lookup"><span data-stu-id="128b6-137">Again, the PostBackTrigger element can specify any control that is a child of any UpdatePanel control in the current unit of encapsulation.</span></span>
