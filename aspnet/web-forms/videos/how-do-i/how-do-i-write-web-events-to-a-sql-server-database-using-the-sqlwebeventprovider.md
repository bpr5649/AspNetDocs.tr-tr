<a name="how-do-i-write-web-events-to-a-sql-server-database-using-the-sqlwebeventprovider"></a>[Bunu nasıl yaparım:] Web olayları SqlWebEventProvider kullanarak bir SQL Server veritabanına yazma
====================
tarafından [Chris piksel](https://twitter.com/chrispels)

Bu video Chris piksel içinde bir web sitesinde bir SQL Server veritabanı hataları günlüğe kaydetmek için ASP.NET Durum İzleme SqlWebEventProvider kullanmayı gösterir. İlk olarak, rol sağlayıcısı ve ASP.NET sistem durumu izleme olayları hakkında bilgi edinin. Ardından, kayıt sistem durumu aspnet kullanarak olayları izleme için gerekli nesneleri ile bir SQL Server veritabanı yapılandırma Bkz\_regiis yardımcı programı, ASP.NET üyelik yapılandırmak için kullanılan aynı yardımcı programı. Ardından, sistem durumu izleme olayları bir ASP.NET web sitesinde yeni oluşturulan SQL Server veritabanına kaydetmek için web.config dosyasındaki yapılandırmayı öğrenin. Bu yapılandırma işleminin bir parçası olarak nasıl kök web.config dosyasında .NET Framework 2.0 sağlayıcıları hem sistem durumu izleme yapılandırırken yararlanılabilir tanımlı olaylar olup. Bu temel özel bilgileri bir SQL Server veritabanını bir ASP.NET web sitesinde oturum sırasında özel olaylar oluşturulabilir temel ' dir.
