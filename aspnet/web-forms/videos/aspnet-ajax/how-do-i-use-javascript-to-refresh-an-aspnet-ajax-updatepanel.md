<a name="how-do-i-use-javascript-to-refresh-an-aspnet-ajax-updatepanel"></a>[Bunu nasıl yaparım:] JavaScript, ASP.NET AJAX UpdatePanel yenilemek için kullanılır?
====================
tarafından [ALi Stagner](https://github.com/JoeStagner)

Bu videoda, biz bir ASP.NET AJAX UpdatePanel denetimine içeriğini yenilemek için JavaScript mantığının kullanmanın iki farklı yollarını öğrenin. İlk yol, bir düğme türü sunucusu denetimi için UpdatePanel ekleyin ve zaman uyumsuz geri göndermenin tetiklemek ve 'geri gönderme UpdatePanel içinde sunucu denetimi tarafından kaynaklandı anlatabilirsiniz' JavaScript kullanmaktır. İkinci yol UpdatePanel için eklenecek herhangi bir düğme türü sunucu denetimi gerektirmez, ancak yine de JavaScript mantığının UpdatePanel içeriğini yenileme kullanmasını sağlıyor. (İkinci teknik Dave Atla tarafından önerilen ve kendi Web günlüğü girişinde "kolayca JavaScript kullanarak UpdatePanel, Yenile" daha fazla bilgi alabilirsiniz.)
