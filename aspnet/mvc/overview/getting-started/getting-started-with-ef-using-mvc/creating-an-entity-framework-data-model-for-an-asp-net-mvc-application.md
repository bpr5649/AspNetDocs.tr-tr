<span data-ttu-id="1e178-315">*Kurallar*nedeniyle en az bir veritabanı oluşturmak için Entity Framework yazmanız gereken kod miktarı veya Entity Framework varsayımlar sağlar.</span><span class="sxs-lookup"><span data-stu-id="1e178-315">The amount of code you had to write in order for Entity Framework to be able to create a complete database for you is minimal because of *conventions*, or assumptions that Entity Framework makes.</span></span> <span data-ttu-id="1e178-316">Bunlardan bazıları zaten belirtilmiştir veya farkında olmadıklarınız olmadan kullanılmıştır:</span><span class="sxs-lookup"><span data-stu-id="1e178-316">Some of them have already been noted or were used without your being aware of them:</span></span>
